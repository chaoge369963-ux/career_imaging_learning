# Codex 教学规则

这个仓库用于成像算法 / AI for Imaging 学习副线。Codex 做代码时必须遵守下面规则。

## 范围

- 默认只修改 `career_imaging_learning` 目录内的文件。
- 不要修改 `RL_SNN3` 主线代码，除非用户明确要求。
- 新内容优先放入对应学习章节目录，不做大而全的项目生成。

## 代码风格

- Python 脚本尽量兼容 Python 3.8。
- 给初学者写代码：少封装、少技巧、变量名直白。
- 每个脚本必须有中文注释，解释关键步骤和图像处理概念。
- 不写复杂工程框架，先保证能运行、能看懂、能复现。

## 教学节奏

- 每次只推进一个小目标。
- 每个脚本都要能输出可检查的结果，例如图片、曲线或打印 shape。
- 每个阶段配一个 README，说明项目目标、输入输出、运行命令和学到的概念。
- 不运行长时间训练，除非用户明确要求。

## 输出约定

- 结果图放到本章节的 `outputs/` 子目录。
- 临时实验文件不要放到仓库根目录。
- 运行命令写在 README 里，方便用户复习。
